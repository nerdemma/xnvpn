Features
Everything you need.
Nothing you don't.

Built from first principles — no bloated daemons, no proprietary protocols, no black boxes.

Layer 3 TUN interface

Captures and injects raw IP packets via kernel tun0 / tun1 — no userspace overhead.

UDP transport

SOCK_DGRAM sockets eliminate TCP Meltdown. Low latency, high throughput.

AES-256-GCM AEAD

Every packet ships with a 12-byte IV and 16-byte GCM tag. Tamper-proof by design.

Bash automation

Interfaces come up automatically via ip addr and ip link. One command deploy.

OpenSSL EVP API

Built on the battle-tested EVP layer. No homebrew crypto. Standard, auditable primitives.

Home server routing

Route traffic through your own machine. Exit from an IP address you trust completely.

How it works
Secure in four steps
01

Traffic captured

Your app generates packets. The kernel routes them into the tun0 virtual interface.

02

Encrypted

xnvpn wraps each packet in AES-256-GCM with a unique IV. Nobody can read it in transit.

03

Tunneled via UDP

The ciphertext travels over a raw UDP socket to your home server. Fast, no overhead.

04

Exits clean

Your server decrypts, verifies the tag, and forwards to the internet from a trusted IP.
